codelikethis
============

source for Code Like This web site

1. git clone
2. cd codelikethis
3. bundle install
4. rspec spec/
5. rake run
6. open http://localhost:9292

---

SCSS currently built using Sass. Run this all the time in development, or at least once before deploy:

    rake build
