# Request Methods

* GET
  * default for links
  * no body
  * parameters passed in query string
* POST
  * default for forms
  * parameters passed in body

# More Request Methods

* PUT
  * a lot like POST
* HEAD
  * same as GET but skips body
  * used to ask if content has changed or expired
* DELETE
  * only used by APIs
